This course will teach you to develop complicated games using procedurally generating techniques. With the use of prefabs and randomness, this course will discuss the creating of two puzzle games: Concentration and Minesweeper.

The first two sections of the course comprise of creating a concentration game, the procedural generation along with logic addition for the Minesweeper game, and finally the last part that covers the steps required to build, host, and share a Unity game using WebGL.
